#  Credit Risk Analysis & Interactive Dashboard

## Overview

This project analyzes customer credit behavior to identify key factors contributing to credit card default risk. Using Python for exploratory data analysis and Tableau for dashboard development, the project provides actionable insights to support data-driven lending and risk management decisions.

---

## Business Problem

A digital lending institution observed increasing delayed repayments and rising credit risk. The objective was to identify high-risk customer segments and uncover patterns that contribute to default behavior.

---

## Dataset

**Source:** UCI Credit Card Dataset

**Records Analyzed:** 30,000 Customers

### Key Attributes

* Age
* Gender
* Education

* Default Status

---

## Key Insights

* Payment history emerged as the strongest predictor of default risk.
* Approximately 22.12% of customers were classified as defaulters.
* Customers with delayed repayments demonstrated significantly higher default probability.
* Customer segmentation enabled identification of high-risk groups for targeted monitoring.

---

## Dashboard Features

* Customer Default Distribution
* Payment Behavior Analysis
* Credit Limit vs Default Analysis

* Interactive Filters

---

## Business Recommendations

* Strengthen approval checks for customers with poor repayment history.
* Implement early warning systems for high-risk segments.
* Use customer segmentation to support risk-based lending strategies.
* Monitor payment behavior proactively to reduce future credit losses.
